Q and A

Bagaimana cara mendaftar akun?

Cukup pergi ke Menu lalu pilih Register. Isi fomulir yang diberikan lalu Register. Sementara untuk mengakses tampilan user. Anda memerlukan izin dari admin terlebih dahulu.

Apa itu SisInfoAlumni?

SisInfoAlumni merupakan sebuah sistem dokumentasi Alumni. Disini kalian bisa mencari alumni, berbicara ataupun berdikusi, dan banyak lagi.

Bagaimana cara menambahkan data saya?

Untuk menambahkan data anda di Sistem ini. Anda diharuskan untuk Login(memiliki akun). Setelahnya anda bisa mengklik tombol menu dipojok kanan. Pilih nama anda lalu pilih tombol partisipasi.

Bagaimana cara saya mengedit data yang salah?

Jika anda tahu data yang benar klik tombol Report dan pilih request edit data, setelahnya admin kami akan menvalidasi data tersebut. Jika data lolos validasi kami akan segera memperbaruinya.

Saya tidak ingin data saya ada disini!

Sebenarnya website ini dirancang untuk membiarkan user menambahkan banyak data sekaligus. Tentu dengan validasi dari Admin. Jika validasi lolos maka akan admin tampilkan. Bila tidak, maka permintaan penambahan user akan dihapus. Namun, bila anda tidak ingin data anda disini. Anda bisa langsung menghubungi kami dari Discord yang berada di sistem. Ingatlah untuk ke channel request terlebih dahulu. Anda tentu juga bisa DM Instagram Saya.

Apakah aman untuk login discord?

Kami menggunakan Api untuk menampilkan discord di halaman ini. Login menggunakan akun discord anda untuk dikusi dapat kami pastikan aman. Karena anda sendiri bisa melihat izin apa yang dibutuhkannya. Kenapa harus login? untuk keamanan privasi. Kami mengharuskan login. Kami juga mengharapkan untuk anda tidak membahas hal privasi di tempat dikusi.

Kenapa diskusi alumni tidak berkerja?

Jika diskusi alumni tidak berkerja atau tidak tampil seperti yang seharusnya. Sihlakan gunakan browser Chrome. Bila masih tidak bisa, mungkin anda berada di mode Icognito atau cookie/localstorage anda mati. Sihlakan aktifkan dengan cara Ini. Untuk browser yang memiliki Adblock/Semacamnya seperti Brave shields. Harap dimatikan terlebih dahulu Opera dan Brave. Bila tidak ada cara diatas yang berkerja pada browser yang anda gunakan sihlakan hubungi kami.

Saya menemukan bugs/error. Apa yang harus saya lakukan?

Jika anda menemukan bugs ataupun error. Kami berharap anda bisa langsung melaporkannya kepada Saya Atau ke channel discord bugs reports di dalam sistem. Dengan melaporkan celah ataupun error anda sudah berpastisipasi dalam meningkatkan keamanan dan kenyamanan dari Sistem Informasi Alumni.

Thank You!
